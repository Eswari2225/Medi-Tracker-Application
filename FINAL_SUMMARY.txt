🔧 QUICK REFERENCE COMMANDS

    Start Everything:
    $ python run.py

    Install Dependencies:
    $ pip install -r backend/requirements.txt
    $ npm install --prefix frontend

    Test Backend:
    $ curl http://localhost:5000/health

    Deploy Prep:
    $ prepare-render.bat          (Windows)
    $ bash prepare-render.sh      (Mac/Linux)

    Clear Cache:
    $ Remove-Item backend\__pycache__ -Recurse -Force
    $ Remove-Item frontend\node_modules -Recurse -Force

🆘 TROUBLESHOOTING

    Problem: Backend won't start
    Solution: Check Python 3.8+, reinstall requirements.txt

    Problem: Frontend won't connect
    Solution: Verify backend running at http://localhost:5000/health

    Problem: Port already in use
    Solution: Kill old process or change PORT environment variable

    Problem: Database error
    Solution: Delete backend/meditracker.db and restart

    Need more help? → Read QUICK_START.md or DEPLOYMENT_GUIDE.md
